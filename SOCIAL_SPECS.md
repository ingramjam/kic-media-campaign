# KIC Campaign - Social Media Specifications

## Quick Platform Guide

### Instagram Reels
- **Size:** 1080×1920px (9:16)
- **Duration:** 15-60 seconds
- **Format:** MP4/MOV
- **Max:** 100MB
- **Captions:** Required

### Instagram Stories
- **Size:** 1080×1920px (9:16)
- **Duration:** 5 seconds (auto)
- **Format:** MP4/MOV or JPG/PNG
- **Max:** 100MB video, 30MB image
- **Stickers:** Polls, sliders recommended

### Instagram Feed
- **Size:** 1080×1080px (square) or 1080×1350px (vertical)
- **Format:** JPG/PNG
- **Max:** 30MB

### Facebook Feed (Video)
- **Size:** 1080×1350px or 1200×628px
- **Duration:** 1-120 seconds
- **Format:** MP4/MOV
- **Max:** 100MB
- **Captions:** Highly recommended (80% watch muted)

### Facebook Feed (Image)
- **Size:** 1200×628px or 1080×1350px
- **Format:** JPG/PNG
- **Max:** 30MB

### Facebook Stories
- **Size:** 1080×1920px (9:16)
- **Duration:** 5-15 seconds
- **Format:** MP4/MOV or JPG/PNG
- **Max:** 100MB video, 30MB image

### Google Search Ads
- **Headline 1:** 30 characters
- **Headline 2:** 30 characters
- **Headline 3:** 30 characters (optional)
- **Description 1:** 90 characters
- **Description 2:** 90 characters

### Google Display Ads
- **Sizes:** 300×250, 728×90, 320×50, 970×90px
- **Format:** JPG/PNG/GIF
- **Max:** 150KB
- **Animation:** Max 30 seconds

### Retargeting Display Ads
- **Sizes:** 300×250, 728×90, 320×50, 970×90px
- **Format:** JPG/PNG/GIF
- **Max:** 150KB

## File Naming Convention

```
{PLATFORM}_{AUDIENCE}_{AESTHETIC}_{TOPIC}_{THEME}_{VERSION}.ext

Examples:
IG_GenZ_GenZ_Vaping_Testimonial_v1.mp4
FB_Millennial_90s_Vaping_Money_v2.mp4
IG_Boomer_Corporate_Smoking_NewYears_v1.jpg
```

**Legend:**
- **PLATFORM:** IG | FB | GG | RT
- **AUDIENCE:** GenZ | Millennial | Boomer
- **AESTHETIC:** GenZ | 90s | Corporate
- **TOPIC:** Vaping | Smoking
- **THEME:** Testimonial | NewYears | Money

## Export Guidelines

### Video Compression (Instagram Reels - Optimal)
- Codec: H.264 (AVC)
- Bitrate: 5,500 Kbps
- Resolution: 1080×1920px
- Frame Rate: 30 FPS
- Audio: AAC, 48kHz, 192kbps stereo

### Video Compression (Facebook - Balanced)
- Codec: H.264
- Bitrate: 3,500-4,000 Kbps
- Resolution: 1080×1350 or 1200×628px
- Frame Rate: 30 FPS
- Audio: AAC, 48kHz, 128kbps stereo

### Image Compression
- **Instagram:** JPG (85-90% quality) or PNG, 150-300KB
- **Facebook:** JPG (80% quality), 200-400KB
- **Retargeting:** JPG (75% quality), 50-100KB

**Tools:** TinyImage, ImageOptim, Photoshop "Save for Web"

## Pre-Upload Checklist

**Videos**
- [ ] Correct resolution (1080×1920 for Reels/Stories)
- [ ] File format correct (MP4/MOV)
- [ ] Duration within limits
- [ ] Captions burned-in or SRT ready
- [ ] Audio normalized (-6dB to -3dB peak)
- [ ] No strobing/flashing
- [ ] Brand colors/logos visible
- [ ] CTA clear and visible
- [ ] File named per convention
- [ ] File size <100MB

**Images**
- [ ] Correct resolution for platform
- [ ] Aspect ratio correct
- [ ] File format correct (JPG/PNG)
- [ ] File size <30MB
- [ ] Color mode RGB
- [ ] Text readable at mobile size
- [ ] High contrast
- [ ] Alt text prepared
- [ ] File named per convention

**Landing Pages**
- [ ] Mobile + desktop tested
- [ ] Forms working (test submission)
- [ ] Pixels installed
- [ ] Copy reviewed
- [ ] Load time <3 seconds
